Email Notification Task Manager
A Flask-based Task Manager Web Application that allows users to create tasks with scheduled times and receive email notifications when the task time arrives.

This project demonstrates how to build a full backend application using Python Flask, PostgreSQL, and SMTP Email Service.

Features
User Registration and Login

Secure User Authentication

Personal Dashboard for each user

Create tasks with scheduled date and time

Email notification when task time arrives

PostgreSQL database integration

Environment variable security using .env

Tech Stack
Backend
Python
Flask

Database
PostgreSQL
SQLAlchemy ORM

Email Service
SMTP (Gmail)

Other Tools
Python-dotenv
HTML
CSS

**Installation Guide**
Clone the Repository

git clone https://github.com/YOUR_USERNAME/Email-Notification.git
cd Email-Notification
Create Virtual Environment

python -m venv venv
Activate virtual environment.

Windows


venv\Scripts\activate
Mac/Linux


source venv/bin/activate
Install Dependencies

pip install -r requirements.txt
